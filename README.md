# Website Văn phòng Thừa phát lại

Website chuyên nghiệp cho Văn phòng Thừa phát lại, được xây dựng bằng React + TypeScript + Vite.

## Tính năng

- ✅ Thiết kế hiện đại, chuyên nghiệp
- ✅ Responsive trên mọi thiết bị
- ✅ Tích hợp Zalo & Phone deep links
- ✅ Google Maps với chỉ đường
- ✅ Form liên hệ với validation
- ✅ Animations mượt mà
- ✅ SEO optimized

## Cài đặt

```bash
# Cài đặt dependencies
npm install

# Chạy development server
npm run dev

# Build cho production
npm run build

# Preview production build
npm run preview
```

## Cấu trúc thư mục

```
src/
├── components/     # React components
├── data/          # Content data
├── hooks/         # Custom React hooks
├── types/         # TypeScript types
├── utils/         # Utility functions
└── index.css      # Global styles
```

## Cập nhật nội dung

Chỉnh sửa file `src/data/content.ts` để cập nhật:
- Thông tin liên hệ
- Dịch vụ
- Đội ngũ
- Tin tức
- FAQ

## Công nghệ sử dụng

- React 18
- TypeScript
